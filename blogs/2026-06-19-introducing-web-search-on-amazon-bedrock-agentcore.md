Web Search on Amazon Bedrock AgentCore is now generally available, enabling AI agents to access current information without relying solely on training data such as real-time stock prices or recent announcements. It features a purpose-built web index spanning tens of billions of documents that refreshes within minutes, a knowledge graph for high-confidence facts, and semantic snippet extraction optimized for model context. Queries remain within AWS infrastructure, and the tool integrates through the Model Context Protocol for discovery and invocation by agents.
