# Amazon Translate

Amazon Translate is a neural machine translation service that delivers fast, high-quality, affordable, and customizable language translation for applications and content.

## APIs

- **Amazon Translate REST API** - RESTful API for Amazon Translate machine translation operations including real-time translation, batch translation jobs, custom terminology, and parallel data management.
